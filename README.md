## ALiaksandr Khalupka

### Geting started

1. clone this repository: `git clone https://github.com/Alex27Khalupka/Go-course-final`

2. create database.

3. set the env var "POSTGRES_URI" - created database url

4. run `make`

5. run `make migrations`

6. run `./apiserver`

#### Docker

To run the application: `docker-compose --build`
